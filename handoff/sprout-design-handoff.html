Design Rules

✅ Always do this

  • Import ALL colours from tokens/theme.ts
  • Import ALL spacing from spacing[n]
  • Use radius.full for ALL buttons
  • Use radius.lg for ALL text inputs
  • Use radius.xl for cards and modals
  • App background = colors.secondary[500] (#f4f1ea cream)
  • Primary text = semanticColors.content.primary
  • Add bottomInset={34} on NavigationBar for iPhone
  • Use typography.fonts.semiBold for button labels
  • Use PlayfairDisplay for ALL headings (h1–h6)
  • Match Figma variant name EXACTLY in props

❌ Never do this

  • Hardcode #186338 anywhere — use the token
  • Hardcode padding like padding: 16 — use spacing[16]
  • Use borderRadius: 8 on buttons — use radius.full
  • Use system fonts (Arial, Helvetica, system-ui)
  • Use dark background (#1a1a1a) as app background
  • Write inline styles with magic numbers
  • Skip the label on an Input — always show it
  • Use opacity for disabled state directly — let the component handle it
  • Implement your own button from TouchableOpacity — use the Button component
  • Forget accessibilityRole on navigation items
